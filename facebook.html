About Facebook OAuth Scopes

Facebook OAuth Scopes are the permissions your app requires from a user. These scopes are displayed to the users (facebook users) to grant your app access to their account. Inside the hood, these permissions grant your app access to the facebook APIs listed below. With these APIs you read or make changes to users' account. For example, with Facebook API you can create a post on behalf of a user. Learn more here

How to Use Our Facebook OAuth Scopes' Link Generator

To generate a link, simply select the scopes you require. All scopes here are written as it is in the Facebook OAuth Scopes documentation. Hover over the question mark icon to see more info on that scope
